# Latido Partners 
## Aplicación de autoevaluación de trabajadores mediante cuestionarios 

## Responsive Design: principalmente utilizada para el entorno movil adaptable para cualquier tipo de pantalla

## Technologias
## Front end
  * Angular 17
  * Angular Material Design
  * TypeScript
  * Bootstrap

## Back end
  * Java 17
  * Spring boot

## Base de datos 
  * PostGreSQL
